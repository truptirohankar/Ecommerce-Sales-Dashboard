# Ecommerce-Sales-Dashboard

Overview :
This Power BI dashboard design to analyze ecoomerse sales data which provides insights into ecommerce sales, customer behavior, product performance, and revenue.

Dashboard:
- Sales Profit by Month
- Product category performance
- Segmentation by Payment Mode
- Profitability and revenue insights
- Filter by Quarter and State

Dataset :
Used CSV file dataset
